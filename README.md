**Financial Instruments Explained** is an educational website that I **designed and developed fully on my own**. The website helps students, beginners, and finance enthusiasts understand financial instruments, asset classes, and related concepts in a clear and interactive way.  

Key features include:
- Definitions and examples of **cash instruments, derivatives, equity, debt, and forex**  
- Historical context on the **development of stock exchanges**  
- Structured **table of contents for easy navigation**  
- Fully **static HTML/CSS**, responsive and lightweight  
- Content sourced from **Investopedia and Wikipedia**, presented in a visually clear way  

This project demonstrates my **front-end development skills, content architecture, and ability to deliver educational tools** for web audiences.
